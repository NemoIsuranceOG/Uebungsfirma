Datenschutz

Diese Website ist eine Projektwebsite einer ACT-Übungsfirma. Sie fragt keine Passwörter, PINs, Bankzugänge, Kreditkartendaten oder CVCs ab. Es findet keine echte Online-Zahlung statt.

Welche Daten werden verarbeitet?

Wenn du das Antrags- oder Kontaktformular nutzt, werden die von dir eingegebenen Formularfelder verarbeitet, zum Beispiel Name, E-Mail-Adresse, Telefonnummer, Adresse, Produktwunsch und Gerätedaten.

Formularübermittlung

Die Formulare verwenden FormSubmit, um die Angaben als strukturierte E-Mail an nemo3018@uebungsfirmen.at zu senden. Es werden keine geheimen API-Schlüssel im Frontend gespeichert.

Hosting

Die Website ist für GitHub Pages vorbereitet. Beim Aufruf einer GitHub-Pages-Seite können technische Zugriffsdaten durch GitHub verarbeitet werden, etwa IP-Adresse, Zeitpunkt des Zugriffs und Browserinformationen.
